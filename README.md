### 💻 Frontend Architecture
* **React.js:** Powers the dynamic, single-page application (SPA) architecture.
* **Material-UI (MUI):** Provides accessible, responsive, and highly customizable UI components to create a polished, app-like interface.
* **Vite:** Utilized as the build tool for lightning-fast Hot Module Replacement (HMR) during development and highly optimized production builds.

### ⚙️ Backend & APIs
* **Node.js & Express.js:** Serves as the robust foundation for the REST API, securely handling user authentication, test evaluations, and database operations.
* **AWS Lambda:** Serverless computing utilized for offloading resource-heavy background processes securely and cost-effectively.

### 🗄️ Database & Cloud Storage
* **MongoDB (Mongoose):** A scalable NoSQL database schema designed to efficiently store and relate user profiles, exam questions, and historical test attempts.
* **AWS S3:** Provides highly available and secure object storage for hosting exam media, user assets, and static platform images.

### 💳 Integrations & Deployment
* **Razorpay:** Integrated RBI-compliant payment gateway for seamless, secure purchasing of premium mock test packages.
* **Vercel:** Edge-optimized continuous deployment network hosting the frontend application.

## 🎯 Project Overview for Recruiters

This project was built to solve a real-world problem: providing accessible, affordable, and high-quality exam preparation for students in Maharashtra who primarily rely on mobile devices.

**Why this stands out:**
1. **End-to-End Ownership:** I independently architected, developed, and deployed the entire stack—from database design to cloud infrastructure to the UI layer.
2. **Production-Grade Features:** Real-world implementations of JWT/OAuth security, secure payment webhooks, and AWS serverless computing.
3. **Problem-Solving Mindset:** Deliberately chose a mobile-first approach to directly address the target audience's hardware limitations.

---

## ✨ Core Features

* 📱 **Mobile-First UI/UX:** Intuitively designed to handle unstable network conditions and smaller screens gracefully.
* 📝 **Mock Tests & PYQs Engine:** A robust, time-bound testing environment simulating real exams, complete with instant result calculation and detailed analytics.
* 🔐 **Secure Authentication:** Multi-layered security using JWT-based email login alongside Google OAuth.
* 💸 **Automated Purchasing:** Frictionless onboarding and test-package unlocking powered by Razorpay webhooks.
